## 🚀 Featured Projects

### 🏛️ [SQL Data Warehouse — Medallion Architecture](https://github.com/singhsneha10/data-engineering-portfolio/tree/main/sql-data-warehouse-project)
> Full Bronze → Silver → Gold medallion data warehouse built on SQL Server

Built a production-style data warehouse that ingests raw CRM and ERP CSV data, cleans and standardises it across layers, and models it into a **Star Schema** for analytics. Includes stored procedures with error handling, timing instrumentation, and dedicated quality check scripts at every layer.

`SQL Server` `T-SQL` `Star Schema` `Stored Procedures` `Data Modelling` `Quality Checks`

---

### 🤖 [ClinIQ — AI Medical Chatbot](https://github.com/singhsneha10/ClinIQ)
> RAG-powered medical assistant grounded in the Gale Encyclopedia of Medicine

Built a **Retrieval-Augmented Generation** chatbot that answers medical questions by searching a FAISS vector index of a trusted medical encyclopedia before generating responses with Llama 3.1 via Groq. Features a full Streamlit UI with multi-session chat history, search, and download.

`LangChain` `FAISS` `Groq (Llama 3.1)` `HuggingFace Embeddings` `Streamlit` `RAG`

---

### 🔄 [CSV to PostgreSQL ETL Pipeline](https://github.com/singhsneha10/data-engineering-portfolio/tree/main/etl)
> Extract → Transform → Load pipeline from SQL Server to PostgreSQL

Automated ETL pipeline that extracts AdventureWorks dimension and fact tables from SQL Server, transforms them with Pandas, and loads them as staging tables into PostgreSQL using SQLAlchemy.

`Python` `Pandas` `SQLAlchemy` `PostgreSQL` `pyodbc` `ETL`

---

### 🕷️ [Automated Job Scraper Pipeline](https://github.com/singhsneha10/data-engineering-portfolio/tree/main/webscraping)
> Scheduled pipeline that fetches live job listings and stores them in PostgreSQL

Scrapes live job data from the Himalayas API on a daily schedule using APScheduler, deduplicates with `ON CONFLICT DO NOTHING`, and persists clean timestamped records to PostgreSQL. Clean modular architecture: config, database, scraper, and scheduler separated.

`Python` `PostgreSQL` `APScheduler` `psycopg2` `REST API` `Scheduling`
